# 学生信息管理系统（一卡通数据分析系统） python1002

## 项目概述

学生信息管理系统是基于 Flask 框架开发的，主要用于一卡通数据分析。系统前端采用 HTML 和 CSS 实现，后端使用 Python 语言开发，数据库选用 MySQL。

## 技术栈

- 前端：HTML + CSS
- 后端：Python
- 数据库：MySQL

## 功能模块

- 学生信息管理
- 一卡通数据查询
- 数据分析

## 系统角色

- 系统管理员
- 教师用户
- 学生用户

## 运行环境

- Python 环境
- MySQL 数据库环境

## 部署步骤

1. 配置 Python 运行环境。
2. 安装依赖库。
3. 初始化 MySQL 数据库。
4. 运行系统。

## 目录结构

```
project/
│
├── app/                # Flask 应用目录
│   ├── templates/      # HTML 模板目录
│   └── views/          # 视图文件目录
│
├── static/             # 静态资源目录（如 CSS 文件）
│
├── venv/               # Python 虚拟环境目录
│
└── migrations/         # 数据库迁移目录
```
